
# Cascade QF Mockup

This is a full-featured mockup React app for exploring a cascade funding model using Quadratic Funding. Built with:

- React 18
- Vite
- Tailwind CSS
- lucide-react

## Running Locally

```bash
npm install
npm run dev
```
